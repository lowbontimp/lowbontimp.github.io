A tool accessing the IRIS data

stp-iris.pl is a perl script for downloading seismic waveform data in the format of SAC from IRIS Web Services on the command-line interface.

A tool accessing the Big Data Open Platform of KIGAM

stp-kigam.pl is a version modified from stp-iris.pl for the Big Data Open Platform of Korea Institute of Geoscience and Mineral Resources.

Perlsac::rwsac

A perl library for reading and writing the SAC file.
